who we are

It is a web portal to understand basics of Machine Learning. It helps to understand data preprocessing, Supervised machine learning, Unsupervised Machine Learning, Feature Engineering, Feature selection and Natural language processing. It allows user to upload datasets and to be pre processed. It creates Machine Learning Model on Given datasets. It Extracts relevant features from datasets and also tranform Features. Natural languge Processing model is also constructed.
